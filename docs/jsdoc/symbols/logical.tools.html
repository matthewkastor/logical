<static> {Boolean} logical.tools.testTruthTable(truth, logicalFunction)
Tests the output of the given logical function against the given truth table. 
 var result = false;
 try {
     result = logical.tools.testTruthTable(
         [
             [0,0,1]
             [0,1,1]
         ],
         logical.or
     );
     if(result) { console.log('success!'); }
 } catch (e) {
     console.log(e.data);
 }
 // logs { arguments: [ false, false ], expected: true, actual: false }
 var result = false;
 try {
     result = logical.tools.testTruthTable(
         [
             [0,0,0]
             [0,1,1]
         ],
         logical.or
     );
     if(result) { console.log('success!'); }
 } catch (e) {
     console.log(e.data);
 }
 // logs "success!"
Parameters:
{Array} truth
The truth table. The rows are arranged with the inputs on the left and the rightmost value being the expected output. For testing logical and, the input 1,1 would be expected to return 1, so the row for this input would be 1,1,1. For more examples see the tests.
{Function} logicalFunction
The logical function to test.
Throws:
{Error}
Throws an error on the first row that doesn't meet expectations. The error will contain a property named "data" containing the failed rows "args", "expected", and "actual" values.
Returns:
{Boolean} Returns true if everything checks out.

<static> {Array} logical.tools.truthTableToBoolean(truthTable)
Converts values in truth tables to boolean values. 
 var truth = [
     [0, 1],
     [1, 0]
 ];
 truth = logical.tools.truthTableToBoolean(truth);
 console.log(truth);
 // logs
 // [
 //     [false, true],
 //     [true, false]
 // ]
Parameters:
{Array} truthTable
An array of arrays containing truth table values.
Returns:
{Array} Returns the truth table with all values converted to boolean values.
