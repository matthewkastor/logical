Logical Functions in JavaScript

Just released a new node module jam packed with logical functions written in JavaScript

So I was getting back to my roots in electronics engineering, going through some of my textbooks from college while searching for some esoteric information I need for a project. While rummaging through the pages I came across a chapter on Karnaugh mapping and was inspired to check if JavaScript has logical operators for all 16 2-ary boolean functions, it doesn't. While JavaScript does include enough of the logical operators to allow you to build any logic expression you could want, things can get a bit hairy with all those symbols in a single statement. I decided to write up a small library (https://github.com/matthewkastor/logical) that gives actual names to the logical expressions in order to make them easier for people to use. Here, I'll list the functions by number, give the JavaScript version of the logical statement, and list the common aliases of the functions.

In the following JavaScript examples "p", "q", and "r" are variables meant to contain Boolean values.

0-ary : Constants

These functions take 0 arguments.

Function: 0
Prose: "false"
Aliases: false, F, contradiction, Opq
JavaScript: false

Function: 1
Prose: "true"
Aliases: true, T, tautology, Vpq
JavaScript: true

1-ary : Unary Operations

These functions take a single argument "p".

Function: 01
Prose: "identity"
Aliases: identity
JavaScript: p

Function: 10
Prose: "not"
Aliases: not, negation
JavaScript: !p

2-ary : Binary Operations

These functions take two arguments "p" and "q".

Function: 0000
See 0-ary 0 above.

Function: 0001
Prose: "p and q"
Aliases: AND, conjunction, Kpq
JavaScript: p && q

Function: 0010
Prose: "p but not q", "p does not imply q", "if not q, follow p, else false"
Aliases: material nonimplication, abdjunction, Xp, Lpq
JavaScript: p && !q

Function: 0011
Prose: "p"
Aliases: projection of p, proposition p, Ipq
JavaScript: p

Function: 0100
Prose: "not p but q", "p is not implied by q", "if not p, follow q, else false"
Aliases: converse nonimplication, converse material nonimplication, Xq, Mpq
JavaScript: !p && q

Function: 0101
Prose: "q"
Aliases: q, projection of q, proposition q, Hpq
JavaScript: q

Function: 0110
"p or q but not both"
Aliases: XOR, exclusive or, exclusive disjunction, Jpq
JavaScript: (p || q) && (!(p && q))

Function: 0111
Prose: "p or q"
Aliases: OR, disjunction, Apq
JavaScript: p || q

Function: 1000
Prose: "neither p nor q"
Aliases: NOR, joint denial, Xpq
JavaScript: !(p || q)

Function: 1001
Prose: "p equals q"
Aliases:XNOR, exclusive NOR, equality, equals, if and only if, iff, biconditional, material biconditional, Epq,
JavaScript: !((p || q) && (!(p && q)))

Function: 1010
Prose: "not q"
Aliases: Nq, Gpq
JavaScript: !q

Function: 1011
Prose: "p is implied by q", "either p, or not q"
Aliases: converse material implication, converse implication, XNq, Bpq
JavaScript: p || !q

Function: 1100
Prose: "not p"
Aliases: Np, Fpq
JavaScript: !p

Function: 1101
Prose: "p implies q", "either not p, or q"
Aliases: material implication, material conditional, material consequence, implication, implies, conditional, XNp, Cpq
JavaScript: !p || q

Function: 1110
Prose: "either not p, or not q", "not p and q simultaneously"
Aliases: NAND, Dpq, alternative denial
JavaScript: !(p && q)

Function: 1111
See 0-ary 1 above

3-ary

These functions take three arguments "p", "q", and "r"

Function: 01000111
Prose: "p if q, else r"
Aliases: conditioned disjunction, then if else
JavaScript: (!q || p) && (q || r)

Function: 01010011
Prose: "if p, follow q, else r"
Aliases: conditioned disjunction, if then else
JavaScript: (p && q) || (!p && r)
